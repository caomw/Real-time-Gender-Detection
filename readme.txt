About the demos
Run FaceDetection.m for a real-time Gender detection Demo. (PS: Photo button is for real-time gender detection, Picture button is for local image. The demo is successfully tested on Windows PC.)

--------------------
A USB camera (or laptop camera) is required for this demo!
